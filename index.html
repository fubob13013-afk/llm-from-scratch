从零开始理解大语言模型 —— 一个美术生自学 AI 的公开笔记。
每课聚焦一个核心概念，配合可交互的浏览器实验，

📖 关于这个项目

这是一棵老头树（Bob）自学大语言模型原理的学习成果。我是一名设计/美术背景的学生， 最初对 AI 只有一知半解。在自学过程中，我发现大多数教程要么假设读者有计算机学位， 要么停留在"调 API"的层面，很少有人在中间搭桥——用可视化的方式讲清楚底层原理。

于是有了这个项目：一套面向零基础学习者的 LLM 课程。每节课聚焦一个核心概念， 先讲原理 → 举具体例子 → 再给你一个浏览器里就能玩的交互实验。 你不需要提前装 Python、不需要 GPU、甚至不需要懂命令行（这些我们会在第 0 课从头教）。 打开浏览器就行了。

项目持续更新中。目前已完成前置基础（Phase 0）和核心原理（Phase 1）， 后续将陆续加入 PyTorch 代码实现、训练实战等内容。最终目标是： 让每个认真学完的人都能从零训练一个自己的微型 GPT。

🗺️ 课程总览

Phase 0 前置基础 —— 工欲善其事 ✓ 已完成

三节课扫清工具障碍。不要求任何编程经验，从打开终端、认识命令行开始。 只教后续课程「刚好够用」的 Python，不贪多。

⚠️ 零基础必读 如果你没有代码基础，并且不会使用 Python，需要从这三节课开始。 后续 Phase 1 的课程会默认你已经掌握了这些前置知识。跳过它们直接看 Transformer 原理会让你寸步难行。
  • L0.1 命令行入门 终端是什么？cd / ls / python
  • L0.2 Python 速通（上） 变量、列表、for 循环、if 判断
  • L0.3 Python 速通（下） 字典、函数、import、读报错
Phase 1 核心原理 —— Transformer 架构全景 ✓ 已完成

这是整个课程的知识骨架。从「LLM 到底在做什么」开始，逐层拆解 Transformer 的每一个组件。 每节课配有浏览器内交互实验，让你直观感受抽象概念。

  • L1 LLM 做什么？ 预测下一个 token · 自监督学习
  • L2 数据准备 BPE 分词 → Token ID → 滑动窗口
  • L3 词嵌入与位置编码 Embedding 查表 · 正弦位置编码
  • L4 注意力机制 Q/K/V · Softmax · 因果遮罩
  • L5 Transformer Block 多头注意力 + FFN + 残差 + LN
  • L6 训练循环 交叉熵 · 梯度下降 · 反向传播
  • L7 GPT 组装与生成 完整架构 · 自回归解码 · 采样
Phase 2 代码实现 —— 用 PyTorch 逐组件实现 GPT ✅ 已完成

把 Phase 1 的「概念理解」变成「能跑通的代码」。从零写出一个 ~150 行的 GPT-2 小型版， 包含 MultiHeadAttention、TransformerBlock、GPTModel 三个核心类。

  • L8 PyTorch 基础 张量 · nn.Linear · autograd
  • L9 实现 Embedding + 位置编码 nn.Embedding · sin/cos 编码
  • L10 实现注意力机制 Q/K/V 投影 · 因果遮罩 · 多头
  • L11 实现 Transformer Block Attention + FFN + 残差 + LN
  • L12 组装完整 GPT-2 模型 GPTModel 类 · 参数统计 · 前向测试
Phase 3 训练实战 —— 真正训练一个模型 ✅ 已完成

有了模型代码，下一步是喂数据、跑训练、看 loss 下降。使用免费 GPU（Colab / AutoDL）， 不需要自己的硬件。产出是一个自己训练的、能生成中文文本的微型 GPT。

  • L13 数据准备实战 文本 → DataLoader · 训练/验证集
  • L14 训练循环实战 train/eval · 学习率调度 · checkpoint
  • L15 云端训练 Colab 免费 GPU · 监控 loss 曲线
  • L16 文本生成实战 加载 checkpoint · Temperature 采样
Phase 4 理解与优化 —— 从「能跑」到「懂为什么」 ✅ 已完成

训练跑通了，但参数怎么算出来的？为什么这样设计？怎么让训练更快更好？ 这个阶段培养技术判断力。

  • L17 参数计算与模型缩放 参数量公式 · Chinchilla 定律
  • L18 训练技巧 warmup · 梯度裁剪 · 混合精度
  • L19 评估方法 Perplexity · Benchmark · 数据污染
Phase 5 进阶方向 —— 按兴趣选修 ✅ 已完成

通向「离线陪伴机器人」长远目标的延伸路线。包含微调对齐、量化部署、多模态、Agent 四个方向， 每个方向 2~3 课，按兴趣选学。

  • A 微调与对齐 SFT · RLHF/DPO · LoRA/QLoRA
  • B 量化与部署 INT8/INT4 · llama.cpp · 本地推理
  • C 多模态 CLIP · Diffusion+DiT · 语音对话
  • D Agent 与工具使用 Function Calling · RAG · Memory
