## 核心理解（课后）
- **FFN**：逐位置的 2 层 MLP（768 → 3072 → 768），升维 → GELU 非线性 → 降维。Attention 负责跨位置信息混合，FFN 负责逐位置的非线性变换
- **残差连接**：output = x + Sublayer(x)，提供梯度高速公路解决深层网络退化问题。Sublayer 只需学「增量」
- **LayerNorm**：对每个 token 的向量做标准化（μ=0, σ=1）+ 可学习的缩放和平移（γ, β）。防止数值漂移，稳定训练
- **Pre-LN vs Post-LN**：Pre-LN（LayerNorm 在子层前）是现代做法，GPT-2 之后标配
- **完整 Block 结构**（Pre-LN）：x → LN → Attention → +x → LN → FFN → +x
- **导数/梯度**：梯度是指路的工具（山的类比），链式法则连乘导致指数衰减，残差连接的 +1 解决了这个问题
- **损失 vs 梯度**：损失值是终极裁判（判断是否学好），梯度只是指路向导

## 设计洞察
- Attention 和 FFN 的分工明确：一个横向（跨位置），一个纵向（逐位置）
- 残差连接不是锦上添花——没有它，深层网络根本训练不了
- LayerNorm 在 NLP 中取代 BatchNorm 的根本原因是变长序列
