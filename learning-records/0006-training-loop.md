# 第6课学习记录 · 训练循环

## 核心理解

训练 = 重复四步循环：
1. **前向传播**：输入 → 模型 → 预测
2. **计算损失**：预测 vs 正确答案 → 一个数字（交叉熵）
3. **反向传播**：链式法则从损失往回算每个参数的梯度
4. **更新参数**：往梯度的反方向挪一小步（梯度下降）

## 关键概念

- **损失函数（Loss）** = -ln(模型给正确答案的概率)，衡量「错得多离谱」
- **梯度** = 损失对某个参数的导数，告诉你该往哪边调
- **学习率** = 每次更新的步长，太大跳过头太小太慢
- **过拟合** = 训练损失降但验证损失升 → 模型在背答案
- **早停法（Early Stopping）** = 验证损失不降了就停
