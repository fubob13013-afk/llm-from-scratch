# 第7课学习记录 · GPT 组装与文本生成

## 核心理解

### 位置编码（Positional Encoding）
- 为什么需要：没有位置信息，「我 爱 你」=「你 爱 我」
- 怎么做：sin/cos 公式给每个位置生成独一无二的 768 维向量，加到词嵌入上
- 关键性质：相邻位置的编码相似，远距离位置差异大
- 不同频率：低维度捕捉局部差异（波长短），高维度捕捉远距离关系（波长长）

### 完整 GPT 架构
```
Token ID → Embedding + Position → Dropout → ×12 Block → LayerNorm → Linear → Softmax → 概率分布
```

### 文本生成（自回归解码）
- 每次预测一个 token → 接在输入后面 → 再预测下一个
- Temperature：控制「创造力」，T↓ 确定，T↑ 随机
- Top-k/Top-p：过滤低概率噪声
