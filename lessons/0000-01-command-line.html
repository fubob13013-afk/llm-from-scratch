💻 第0.1课 · 命令行入门 — 终端是什么？

这是 Phase 0 的第一课。如果你从来没用过「命令行」或「终端」——没关系，这节课就是为你准备的。我们只学后续课程用得到的 6 个命令。

🎯 本课目标 学完之后，你能在自己电脑上：① 打开终端 ② 导航到指定文件夹 ③ 运行一个 Python 脚本 ④ 用 pip 安装 Python 包。这四项技能是 Phase 2+3 写代码和训练模型的前提。

什么是「终端」？为什么需要它？

你平时用电脑，是通过图形界面（鼠标点击图标、菜单、按钮）。终端（Terminal，也叫命令行 / Command Line）是另一种操作电脑的方式——通过打字来给电脑下指令。

为什么学 AI/编程需要终端？两个原因：

  1. 很多 AI 工具只有命令行版本。训练模型的脚本、安装 PyTorch 的命令、启动 Jupyter Notebook——这些都是在终端里敲一行字完成的。
  2. 命令行是最精确的沟通方式。图形界面每次点击在不同电脑上可能效果不同；但 python train.py 在任何人的电脑上都是「运行 train.py 这个文件」——精确、可复现。
🔑 核心类比 图形界面 = 在餐厅对着菜单指指点点。
终端 = 直接对厨师说「一碗牛肉面，不放香菜」——更快、更精确，但需要你知道怎么点菜。

怎么打开终端？—— 按你的操作系统选择

⚠️ 请先确认你用的是哪个操作系统（Windows / macOS / Linux），然后按照下面对应的方法操作。不要混着看——不同系统的终端命令不一样，混着看会出错。

🪟 Windows 用户

方法一：Win+R（推荐，最通用）

  1. 同时按下键盘上的 Win 键（Windows 徽标键，通常在键盘左下角，Ctrl 和 Alt 之间）和 R 键。
  2. 屏幕左下角会弹出一个叫「运行」的小窗口。
  3. 在输入框里输入 cmd（三个字母，小写），然后按 回车（Enter）。
  4. 弹出一个黑底白字的窗口——这就是命令提示符（CMD），Windows 自带的终端。

方法二：开始菜单搜索

  1. 按一下 Win 键（单独按，打开开始菜单）。
  2. 直接打字输入 cmd 或 PowerShell。
  3. 搜索结果中会出现「命令提示符」或「Windows PowerShell」，点击打开。
💡 CMD 还是 PowerShell？
对初学者来说，选 CMD（命令提示符）就够了。PowerShell 功能更强大但更复杂。本课的命令在 CMD 和 PowerShell 中都能运行——但具体命令不同（详见下文每个命令的说明）。

打开后，你会看到类似这样的文字：

C:\Users\你的用户名>

最后的 > 就是提示符（Prompt）——它在说「我准备好了，请输入命令」。

🍎 macOS 用户

  1. 同时按下 Command (⌘) + 空格，打开 Spotlight 搜索。
  2. 输入 Terminal（或中文输入「终端」）。
  3. 搜索结果中出现「终端.app」，按 回车 打开。

打开后，你会看到类似这样的文字：

你的用户名@你的电脑名 ~ %

末尾的 %（新系统）或 $（旧系统）就是提示符（Prompt）。

🐧 Linux 用户

  1. 同时按下 Ctrl + Alt + T。
  2. 终端窗口会自动弹出。

打开后，你会看到类似这样的文字：

你的用户名@你的电脑名:~$

末尾的 $ 就是提示符（Prompt）。

不管哪种系统，提示符的含义都是一样的：「我在等你的命令——你想让我做什么？」在提示符后面打字，然后按回车，电脑就会执行你输入的命令。

1 你在哪？—— 查看当前位置和文件列表

终端运行时，永远「位于」硬盘上的某一个文件夹里。就像文件管理器有一个「当前打开的文件夹」，终端也有一个当前工作目录（Working Directory）。先搞清楚「我在哪」和「这里有什么」是一切操作的基础。

🪟 Windows（CMD 命令提示符）

查看当前位置： 

C:\Users\bob> cd
C:\Users\bob           ← 在 CMD 里，直接输入 cd（不加任何参数）就显示当前目录

列出当前目录下的文件：

C:\Users\bob> dir
2026/07/03  14:30    <DIR>          Desktop
2026/07/03  14:30    <DIR>          Documents
2026/06/20  09:15              142  hello.py
2026/06/18  16:42    <DIR>          data

在 Windows PowerShell 中，ls 和 pwd 也可以用（它们是 dir 和 Get-Location 的别名），但建议初学者记住原生命令：cd（无参数）= 显示当前位置，dir = 列出文件。

🍎🐧 macOS / Linux

查看当前位置 —— pwd（Print Working Directory）：

$ pwd
/Users/bob              ← 输出：你当前在 /Users/bob 这个文件夹里

这是最安全的命令——不改变任何东西，只是告诉你「你在哪」。

列出当前目录下的文件 —— ls（List）：

$ ls
Desktop                ← 桌面文件夹
Documents              ← 文档文件夹
hello.py               ← 一个 Python 文件
data                   ← 数据文件夹

2 怎么移动？—— 切换目录

切换目录就是「双击文件夹」的终端版本。

🪟 Windows CMD

C:\Users\bob> cd Documents        ← 进入 Documents 文件夹
C:\Users\bob\Documents> cd          ← 无参数：显示当前位置
C:\Users\bob\Documents
C:\Users\bob\Documents> cd ..       ← 两个点表示「上一级」
C:\Users\bob>                       ← 回来了
C:\Users\bob> cd C:\                ← 直接跳到 C 盘根目录（输入完整路径）

Windows 路径用反斜杠 \ 分隔：C:\Users\bob\Documents

🍎🐧 macOS / Linux

$ cd Documents        ← 进入 Documents 文件夹
$ pwd
/Users/bob/Documents   ← 确认：现在在 Documents 里了

$ cd ..               ← 两个点表示「上一级」
$ pwd
/Users/bob             ← 回来了

$ cd ~                ← 波浪号 = 用户主目录（回家）
$ pwd
/Users/bob

macOS / Linux 路径用正斜杠 / 分隔：/Users/bob/Documents

常用路径符号（所有系统通用）

.当前目录（「就在这儿」）
..上一级目录（「退回一步」）
~用户主目录（「回家」）。macOS/Linux 原生支持；Windows CMD 不支持，PowerShell 支持

3 怎么运行 Python？—— python

在终端里输入 python 文件名，就是让 Python 解释器去执行那个文件：

$ python hello.py
你好，世界！           ← 这是 hello.py 运行后打印的输出

前提是：① 你的电脑安装了 Python ② 终端当前在 hello.py 所在的文件夹里（用 cd 导航过去）。

⚠️ 常见问题 如果输入 python 后提示「不是内部或外部命令」，说明 Python 没有正确安装或没有添加到系统 PATH。去 python.org 下载安装时，一定要勾选「Add Python to PATH」（把 Python 加到系统路径）这个复选框。

4 怎么安装 Python 包？—— pip install

Python 自带的功能有限。PyTorch、NumPy、Jupyter 这些需要额外安装。安装命令是 pip install：

$ pip install torch numpy jupyter

一行命令可以同时安装多个包。pip 会自动下载、解压、放到正确的位置。你以后看到任何教程里写 pip install xxx，就知道是在终端里敲这一行。

三个最常用的 pip 命令：

$ pip install 包名         ← 安装
$ pip list                 ← 查看已安装了哪些包
$ pip show 包名            ← 查看某个包的详细信息（版本、位置等）

终端操作小贴士

快捷键 / 技巧作用
Tab自动补全。输入文件名的前几个字母，按 Tab，终端自动补全。大幅减少打字量。
↑ / ↓回溯历史命令。按 ↑ 回到上一条命令，不用重新打。
Ctrl + C终止当前正在运行的程序。如果训练中途想停，按这个。
Ctrl + L 或 clear清屏。屏幕太乱了？一键清空。
文件路径中的空格如果文件夹名有空格（如 "My Documents"），用引号包起来：cd "My Documents"
复制粘贴Windows 终端：Ctrl+C/V 或右键。Mac 终端：Command+C/V。
⚠️ Windows 用户请注意

1. 路径分隔符不同：Windows 用反斜杠 \（例如 C:\Users\bob\project），macOS/Linux 用正斜杠 /（例如 /Users/bob/project）。这是历史原因造成的，知道就好。

2. CMD 命令 ≠ Linux 命令：很多网上教程默认你在用 Mac/Linux，写的命令是 ls、pwd、rm 等。这些命令在 Windows CMD 里不起作用。如果你在 CMD 里输入 ls 看到「不是内部或外部命令」，说明这是 Linux 命令——在 CMD 里换成对应的 Windows 命令（如 dir）。PowerShell 对 Linux 命令的兼容性更好（支持 ls），如果你习惯用 Linux 风格，可以用 PowerShell 代替 CMD。

3. 后续 Phase 2+3 的 AI/机器学习代码建议使用以下方案之一：
  • Git Bash：搜索 "Git for Windows" 下载安装，它会给你一个和本课模拟终端体验一致的命令行环境（支持 ls、pwd 等 Linux 命令）。
  • WSL（Windows Subsystem for Linux）：Windows 内置的 Linux 子系统，AI 社区最推荐的方案。
  • PowerShell：Windows 自带，比 CMD 强大，对本课的命令兼容性也更好。

🧪 小测验

问题 1/3 — 导航

你当前在 /home/user/project。输入什么命令可以回到 /home/user？

点击看答案
cd ..
.. 表示「上一级目录」。project 的上一级是 user。也可以用 cd ~ 直接回家（如果 ~ 指向 /home/user）。

问题 2/3 — 运行脚本

你写了一个 Python 文件叫 train.py，放在当前目录。怎么在终端里运行它？

点击看答案
python train.py
前提：① 终端当前在 train.py 所在的目录（用 ls 确认能看到它）② 电脑已安装 Python。如果系统同时装了 Python 2 和 3，可能需要用 python3 train.py。

问题 3/3 — 安装包

你打开了一个 AI 项目的 README，第一行写着「请先运行 pip install torch numpy」。这条命令应该在哪里执行？它会做什么？

点击看答案
在终端（命令行）里执行。它会从 Python 包仓库（PyPI）下载 torch 和 numpy 两个包，自动安装到你的 Python 环境中。

不需要在 Python 代码里写这行——pip install 是终端命令，不是 Python 代码。
