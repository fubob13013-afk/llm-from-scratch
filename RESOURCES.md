# LLMs from Scratch 学习资源

## 核心教材

- [GitHub: rasbt/LLMs-from-scratch](https://github.com/rasbt/LLMs-from-scratch) ⭐ 主教材
  Sebastian Raschka 著《Build a Large Language Model (From Scratch)》官方配套代码。7 章从文本处理到指令微调，Jupyter Notebook 格式，可在普通笔记本运行。MIT 协议。
- [中文翻译: MLNLP-World/LLMs-from-scratch-CN](https://github.com/MLNLP-World/LLMs-from-scratch-CN)
  上述项目的中文翻译版，代码注释和说明均已汉化。
- [中文翻译: datawhalechina/llms-from-scratch-cn](https://github.com/datawhalechina/llms-from-scratch-cn)
  Datawhale 社区的另一个中文版本，附有组队学习资源。

## 视频课程（仅收录国内可访问的中文资源）

- [B站: 《从零构建大模型》配套视频中文版](https://www.bilibili.com/video/BV1EQtuz2EJM) ⭐ 首选
  Sebastian Raschka 原作的官方中文版视频教程，GitHub 63.7k star。第2章覆盖文本分词、BPE算法、滑动窗口采样等数据准备全流程。
- [B站: 《大模型预训练的数据如何准备》](https://www.bilibili.com/video/BV1mz8GzuEyA)
  Bob 发现的教程，基于 LLMs-from-scratch 讲解数据准备环节。适合作为 Ch2 的辅助观看。
- [B站: 《大模型训练第一步：数据集与分词全解析》](https://www.bilibili.com/video/BV1dAEz6EEEP)（中字）
  短小精悍的分词入门，对比词级/字符级/BPE 三种分词方案。
- [B站: Transformer 最强动画讲解](https://www.bilibili.com/video/BV1fGeAz6Eie) ⭐ 讲注意力机制时必看
  动画可视化讲解 Transformer，从 QKV 矩阵到多头注意力，通俗易懂。适合 Ch3 学习时观看。
- [B站: 李宏毅 2024 生成式人工智能导论](https://search.bilibili.com/all?keyword=李宏毅+生成式人工智能+2024)
  台大李宏毅教授，中文授课，零基础友好。B站有大量搬运，搜索即可。
